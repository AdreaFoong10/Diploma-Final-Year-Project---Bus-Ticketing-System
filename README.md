## About the Project
CometBus is designed to address common issues faced at physical bus stations, such as long queues, overcrowding, and time-consuming ticket purchases.

With the increasing reliance on internet-based services, users now prefer to book tickets online rather than waiting in line. CometBus allows users to **search, book, and pay for bus tickets online**, offering a faster and more convenient alternative to traditional ticket counters.

The system emphasizes:
- Clear and complete information
- Accurate and transparent pricing
- Simple and user-friendly booking flow
- 24/7 accessibility from any location with an internet connection

---

## Project Objectives
- Improve **information completeness**, especially ticket pricing
- Present important information **without excessive scrolling**
- Reduce congestion at bus stations
- Provide a **simple, fast, and transparent** booking experience

## System Features

### Admin
- Login and logout
- View and edit admin profile
- Add and manage staff
- Add, edit, and delete bus schedules
- View customer list and booking details
- Manage bus routes
- Add and edit bus operators
- Add, edit, delete, and view bus drivers
- Add and edit bus information
- View all customer details

---

### User
- Register and log into the system
- View and edit profile
- Search bus tickets based on preferences
- View available bus schedules
- Select seats and purchase bus tickets
- Make online payments
- View booking history
- Reset password using **Forgot Password**
- Logout securely
